# Coding Standards

## Java Naming Convention

Classes

- PascalCase

Example

UserService

DriverController

---

Methods

camelCase

Example

calculateShortestPath()

registerUser()

---

Variables

camelCase

Example

fuelCost

estimatedTime

---

Constants

UPPER_CASE

Example

MAX_DISTANCE

JWT_SECRET

---

Packages

lowercase

Example

controller

service

repository

entity

algorithms

---

REST API

Use nouns instead of verbs.

Good

/api/users

/api/orders

Bad

/getUsers

/createOrder

---

General Rules

- Follow SOLID principles.
- Keep methods small and focused.
- Avoid duplicate code.
- Use meaningful names.
- Write reusable components.
