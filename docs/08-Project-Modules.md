# Project Modules

SmartRoute AI consists of eight major modules.

## 1. Authentication Module

Responsibilities

- Registration
- Login
- JWT Authentication
- Role Management

---

## 2. Customer Module

Responsibilities

- Create Orders
- Track Deliveries
- View History

---

## 3. Driver Module

Responsibilities

- Accept Deliveries
- Navigate Routes
- Update Status

---

## 4. Warehouse Module

Responsibilities

- Manage Warehouses
- Inventory
- Vehicle Allocation

---

## 5. Route Optimization Engine

Responsibilities

- Build Graph
- Run Dijkstra
- Run A*
- Compare Algorithms
- Calculate Fuel Cost

---

## 6. Maps Module

Responsibilities

- Display Routes
- Live Driver Tracking
- Traffic Overlay

---

## 7. Analytics Module

Responsibilities

- Reports
- Charts
- Driver Performance
- Delivery Performance

---

## 8. Admin Module

Responsibilities

- Manage Users
- Warehouses
- Drivers
- System Monitoring
