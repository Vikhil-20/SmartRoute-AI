# Code of Conduct

## Our Standards

- Respect everyone.
- Be professional.
- Provide constructive feedback.
- Encourage learning.
- Collaborate openly.

## Unacceptable Behavior

- Harassment
- Offensive language
- Personal attacks
- Discrimination

## Goal

Create a positive and collaborative learning environment.# Code of Conduct
